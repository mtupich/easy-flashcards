# Easy Flashcards

App de flashcards em SwiftUI para iOS. Permite criar diferentes grupos (baralhos) de flashcards e estudar de forma organizada.

## Tecnologias

- SwiftUI
- Core Data (persistência local)
- Arquitetura MVVM-C

## Funcionalidades

- Login com usuário e senha
- Tela inicial com resumo de baralhos, cartões e progresso
- Grupos de flashcards personalizáveis (ex.: Inglês Básico, Negócios, Viagem)
